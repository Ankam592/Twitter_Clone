# 🧵 ThreadIQ – A Production-Grade Twitter Clone (MERN Stack)

ThreadIQ is a feature-rich, production-ready Twitter clone built using the **MERN stack** (MongoDB, Express.js, React, Node.js), enhanced with modern UI/UX and AI capabilities.

---

## 🚀 Features

### 🔍 Core Functionality
- 📝 Post tweets with optional image and **AI-based speech-to-text** input.
- 🔁 Infinite scroll-style tweet feed with **carousel-like navigation**.
- 👤 User authentication & profile system.
- 📸 Profile picture support for every user.
- 🗑️ Full CRUD operations on tweets.

### 🤖 AI-Powered Capabilities
- 🎙️ **Speech-to-Text** tweet input using Web Speech API.
- ☣️ **Toxic Comment Detection** to warn/block inappropriate tweets using Perspective API.

### 🧠 State Management
- ⚛️ Global state handled via **Redux Toolkit**.
- 🧩 **Custom Hooks** for reusable logic like data fetching and authentication.

### 💅 UI/UX
- 🎨 Fully responsive layout with **TailwindCSS**.
- 🎠 Interactive **carousel and touch-feel cards** for user browsing.
- 🎯 Clean, consistent **blue-themed UI** across the app.

### 🔐 Auth & Security
- 🍪 JWT-based authentication stored in **secure cookies**.
- 🔄 Protected routes using custom **middleware** on both frontend and backend.

---

## 📦 Tech Stack

| Layer       | Tech Used                           |
|-------------|-------------------------------------|
| Frontend    | React.js, TailwindCSS               |
| State Mgmt  | Redux Toolkit                       |
| Backend     | Node.js, Express.js                 |
| Database    | MongoDB (Compass as client)         |
| AI          | Web Speech API, Perspective API     |
| Extra       | react-router, react-icons, custom hooks, animations |
